### 五、面试项目功能清单（建议实现）

#### Phase 1: 核心 MVP（2-3 周）

**1. 工作流编排**
```yaml
功能:
  - 可视化画布（React Flow）
  - 节点类型: 开始/LLM/知识库/条件/工具/结束
  - 边连接: 顺序边 + 条件边
  - 节点配置面板
  - DSL JSON 导入/导出

技术实现:
  前端: React Flow + 表单渲染
  后端: LangGraph 动态编译
```

**2. RAG 知识库**
```yaml
功能:
  - 文档上传（PDF/TXT/MD）
  - 自动切片 + 重叠配置
  - 向量化存储（LanceDB/pgvector）
  - 检索测试（调试用）
  - 绑定到智能体/工作流节点

技术实现:
  切片: LangChain TextSplitter
  Embedding: OpenAI / 国产模型
  检索: 向量相似度 + TopK
```

**3. 智能体基础**
```yaml
功能:
  - 创建/编辑/删除智能体
  - 人设文本 + 模型参数
  - 绑定知识库（0-N 个）
  - 绑定工作流（0-1 个）
  - 对话测试界面

技术实现:
  提示词组装: System + Memory + Context
  流式输出: SSE
```

**4. MCP 技能接入**
```yaml
功能:
  - MCP Server 配置（stdio/SSE）
  - 技能列表展示
  - 动态启用/禁用
  - 工具调用链路

技术实现:
  SDK: mcp-python-sdk
  安全: 沙箱执行或远程服务
```
